# Simple Interest Calculator

## 📌 Project Overview
This project is a Simple Interest Calculator that calculates interest based on user input: principal, rate, and time.

## 📌 Project Name
Simple Interest Calculator

## 📌 Formula Used
Simple Interest (SI) = (P × R × T) / 100

Where:
- P = Principal amount
- R = Rate of interest (% per year)
- T = Time (years)

## 📌 Features
- Accepts user input
- Calculates simple interest automatically
- Lightweight Bash script implementation

## 📌 Example
Input:
- P = 1000
- R = 5
- T = 2

Output:
SI = 100

## 📌 Purpose
This project demonstrates basic shell scripting and arithmetic operations in Linux environment.
